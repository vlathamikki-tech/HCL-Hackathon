# 📊 Sales & Profit Dashboard (Power BI)

## 🔹 Overview
This dashboard analyzes sales, profit, discount, and shipping performance.

## 🔹 Features
- Dynamic metric selection
- Time intelligence (Rolling 12M, Cumulative)
- Discount risk analysis
- Shipping performance insights

## 🔹 Files
- Hackathon - hcl.pbix → Main report
- Sample - Superstore.xlsx → Source data

## 🔹 How to Use
1. Download .pbix file
2. Open in Power BI Desktop

## 🔹 Tools Used
- Power BI
- DAX
- Data Modeling
